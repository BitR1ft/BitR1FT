## Featured Project — UniVex

> An AI-powered, fully autonomous penetration testing platform.  
> Give it one target. It handles the rest.

**UniVex** is the project I'm most proud of. It automates recon, exploitation, post-exploitation, and reporting — powered by 145+ tools, 13 AI agents, and 11 MCP servers in an isolated Kali container.


**Stack:** FastAPI · Next.js · PostgreSQL · Neo4j · LangGraph · Docker  

**Highlights:**
- 145+ tools across web, AD, cloud, proxy, pivoting  
- 12+ LLM providers (GPT-4, Claude, Gemini, Llama 3.3)  
- ~98% bug bounty tool coverage  
- Built-in HTTP proxy (Burp-equivalent)  
- BloodHound AD paths, deserialization chains, tunneling/pivoting
